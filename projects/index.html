Emotion Detection in Messaging

A dual-model NLP system that detects emotions in messages and modifies Signal's UI in real-time.

This system uses RoBERTa-based binary and multi-class classifiers trained on DailyDialog, IEMOCAP, and MELD datasets. The models process current and previous 5 messages, and color message bubbles based on emotion using a custom Flask server and Signal app integration.
